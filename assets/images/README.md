# Image Assets

Place Fraggle Rock images in this directory:

## Recommended Structure

```
images/
├── characters/
│   ├── gobo.jpg
│   ├── red.jpg
│   ├── mokey.jpg
│   ├── wembley.jpg
│   ├── boober.jpg
│   ├── doozers.jpg
│   └── gorgs.jpg
├── episodes/
│   ├── season-1/
│   ├── season-2/
│   └── ...
├── behind-scenes/
│   └── jim-henson.jpg
└── og-images/
    └── social-share.jpg
```

## Image Guidelines

- **Format**: WebP preferred (fallback to JPG)
- **Dimensions**: 
  - Hero images: 1920x1080
  - Character portraits: 800x800
  - Thumbnails: 400x400
- **Optimization**: Compress to under 200KB
- **Alt Text**: Always include descriptive alt text
