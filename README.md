## ğŸš€ Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Basic understanding of HTML/CSS/JavaScript
- Web server for local development (optional)

### Local Development

1. **Clone or download the project**
   ```bash
   cd Fraggle_Rock_Fan
   ```

2. **Open in your browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js with http-server
   npx http-server
   
   # VS Code Live Server extension
   # Right-click index.html â†’ Open with Live Server
   ```

3. **View the website**
   - Navigate to `http://localhost:8000` (or your server's address)
   - The homepage will load with full functionality

### Adding the Theme Song Easter Egg

1. Download a legal copy of the Fraggle Rock theme song (MP3 format)
2. Create the directory: `assets/audio/`
3. Place the file as: `assets/audio/fraggle-rock-theme.mp3`
4. Click the logo on any page to hear the theme!

## ğŸ”§ Customization Guide

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --fraggle-orange: #FF6B35;  /* Change primary color */
    --radish-purple: #7B2CBF;   /* Change accent color */
    /* ... more colors ... */
}
```

### Adding New Pages

1. Create new HTML file (e.g., `new-page.html`)
2. Copy the navigation and footer from an existing page
3. Add the page link to all navigation menus
4. Create corresponding CSS file if needed
5. Update internal links throughout the site

### Adding Images

1. Place images in `assets/images/`
2. Use descriptive filenames (e.g., `gobo-fraggle-portrait.jpg`)
3. Add proper alt text for accessibility
4. Consider using WebP format for optimization
5. Implement lazy loading for performance
